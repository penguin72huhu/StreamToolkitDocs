# Mendapatkan DeepL API Key

DeepL API Key digunakan untuk terjemahan otomatis fitur berikut:
- **Terjemahan suara** — Terjemahan otomatis setelah mengubah suara menjadi teks
- **Chat** — Terjemahan otomatis pesan penonton

## Langkah 1: Login ke akun DeepL Anda

Buka [DeepL](https://www.deepl.com) dan login ke akun Anda. Jika Anda belum memiliki akun, Anda harus mendaftar terlebih dahulu.

## Langkah 2: Masuk ke pengaturan Account

Klik **ikon profil** di sudut kanan atas dan pilih **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Langkah 3: Beralih ke tab API Keys & limits

Klik tab **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Langkah 4: Buat API Key baru

1. Klik **Create key +**
2. **Name your key**: Masukkan nama apa pun (misalnya `Stream Toolkit`)
3. **Permissions**: Pilih **All access**
4. Klik **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Langkah 5: Salin dan tempel ke App

1. Salin API Key yang telah dibuat
2. Kembali ke Stream Toolkit dan tempel ke kolom **DeepL API Key** yang sesuai

![Paste API Key](./deepl-api-img/step5.png)

## Pertanyaan Umum

**Q: Apakah uji coba gratis DeepL memiliki batas penggunaan?**
Ya. Uji coba gratis menyediakan kuota 1.000.000 karakter dan terbatas hingga satu bulan. Untuk penggunaan terjemahan berkualitas tinggi secara berkelanjutan, silakan berlangganan paket berbayar DeepL.

**Q: Apa yang harus dilakukan jika API Key saya bocor?**
Kembali ke DeepL Account → API Keys & limits, hapus Key lama dan buat yang baru.

# Twitch Pengaturan

Stream Toolkit menggunakan otorisasi OAuth untuk menghubungkan akun Twitch Anda, sehingga tidak perlu memasukkan kunci secara manual.

## Langkah 1: Hubungkan akun Twitch

1. Buka Stream Toolkit
2. Klik **Pengaturan** pada menu kiri bawah
3. Temukan **Twitch** di bawah Koneksi Platform dan klik **Hubungkan akun**
4. Browser akan otomatis membuka halaman otorisasi Twitch
5. Masuk dan klik **Otorisasi**

Setelah otorisasi selesai, Stream Toolkit akan otomatis mendapatkan kunci akses yang diperlukan.

## Pertanyaan Umum

**Q: Tidak ada respons setelah otorisasi?**
Pastikan browser Anda tidak memblokir jendela pop-up. Stream Toolkit akan membuka jendela baru untuk menyelesaikan otorisasi.

**Q: Apakah harus menjadi Mitra atau Afiliasi Twitch untuk menggunakannya?**
Tidak. Akun biasa juga dapat menggunakan fitur obrolan. Target langganan/Bits memerlukan fitur langganan yang sudah diaktifkan.

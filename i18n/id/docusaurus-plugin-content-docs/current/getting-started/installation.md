# Instalasi dan Peluncuran

## Persyaratan Sistem

- Windows 10 / 11 (64-bit)
- Koneksi internet

## Unduh dan Instalasi

1. Buka [Halaman Steam](https://store.steampowered.com/app/4828880/Stream_Toolkit/) untuk membeli Stream Toolkit
2. Klik "Instal" di Steam dan tunggu hingga pengunduhan selesai
3. Luncurkan Stream Toolkit dari Steam

## Peluncuran Pertama

Setelah diluncurkan, Anda akan melihat antarmuka kontrol utama, dengan menu kiri yang mencantumkan semua fitur.

Setiap fitur Stream Toolkit memiliki URL lokal yang sesuai (misalnya, `http://127.0.0.1:7272/donate`). Dengan menambahkan "Browser Source" di OBS dan memasukkan URL tersebut, Anda dapat menampilkan layar fitur pada siaran langsung.

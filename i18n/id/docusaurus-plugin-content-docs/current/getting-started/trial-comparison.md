# Perbandingan Versi Uji Coba / Versi Penuh

Stream Toolkit menyediakan **versi uji coba gratis** dan **versi penuh**. Versi uji coba tidak memiliki batas waktu dan memungkinkan penggunaan permanen dari sebagian besar fitur interaktif untuk pengalaman penuh; versi penuh membuka integrasi pagamento, Steam Workshop, kustomisasi tingkat lanjut, dan menghapus berbagai batasan jumlah.

:::info Tentang Peningkatan Sapi
Versi uji coba dan versi penuh adalah iki aplikasi terpisah di Steam. Setelah membeli versi penuh, Anda harus menginstalnya secara terpisah, ve pengaturan serta riwayat di antara iku tidak saling terhubung.
:::

# Stream Toolkit Pengenalan

Menu kiri dikategorikan berdasarkan fungsi. Pemula disarankan untuk mulai membaca dari [Memulai Cepat](./getting-started/installation).

---

Stream Toolkit adalah toolkit streaming serbaguna yang dirancang untuk kreator konten, sehingga Anda tidak perlu lagi berpindah-pindah antar berbagai situs web.

- **Terjemahan real-time**: terjemahan chat dan suara bawaan memudahkan Anda berinteraksi dengan penonton dari seluruh dunia.
- **Semua dalam satu**: polling, giveaway, goal bar, notifikasi donasi, dan Media Board terintegrasi dalam satu aplikasi.
- **Kustomisasi**: mendukung kustomisasi CSS dan tema Steam Workshop.
- **Pengaturan cepat**: buat URL Overlay hanya dengan satu klik dan tempel langsung ke OBS.

---

## 🌍 Terjemahan Chat Real-Time

Gabungkan pesan Twitch dan YouTube dalam satu tempat.

Terjemahkan pesan berbahasa asing secara otomatis dan berkomunikasi lebih mudah dengan penonton di seluruh dunia.

Mendukung avatar, lencana, ikon platform, kustomisasi CSS, dan berbagi tema Steam Workshop.

## 🎙️ Subtitle Terjemahan

Tahan tombol pintas untuk memulai perekaman.

Ucapan Anda akan diterjemahkan ke bahasa yang dipilih dan ditampilkan selama siaran.

## 🎨 Tema Steam Workshop

Unduh tema yang dibuat komunitas atau buat gaya Overlay Anda sendiri.

Sesuaikan tampilan dan animasi chat, goal bar, notifikasi donasi, dan timer.

Mendukung berbagi dan unduhan melalui Steam Workshop.

## 🎮 Semua Alat Streaming dalam Satu Tempat

Fitur yang tersedia:

- Spin Wheel
- Polling Langsung / Giveaway
- Bilah Target Twitch
- Bilah Target Donasi / Notifikasi Donasi / Papan Donasi
- Timer
- Jam
- Permintaan Lagu
- Media Board (Gambar / Video / Suara)
- Dinding Emote
- Penanda Sorotan

Tidak perlu lagi memasang banyak plugin atau membuka banyak alat web.

## ⚡ Dirancang untuk OBS

Setiap fitur memiliki URL OBS tersendiri.Cukup salin dan tempel untuk langsung digunakan.

Chat, goal bar, timer, dan notifikasi donasi dapat berjalan secara independen.

Habiskan lebih sedikit waktu mengelola alat dan lebih banyak waktu untuk streaming.

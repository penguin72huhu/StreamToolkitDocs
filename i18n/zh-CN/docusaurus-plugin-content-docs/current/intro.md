# Stream Toolkit 介绍

左侧菜单按功能分类。新手建议从 [快速开始](./getting-started/installation) 开始阅读。

---

Stream Toolkit 是专为主播打造的万能直播工具箱，告别手忙脚乱地切换多个网页。

- **实时翻译**： 内置聊天室与语音双向翻译，轻松与全球观众互动。
- **全面整合**： 集投票、抽奖、目标条、赞助通知与实况效果板于一体。
- **外观自定义**： 支持 CSS 自定义与 Steam 创意工坊，自由切换风格。
- **快速上手**： 一键生成 Overlay 地址，复制到 OBS 即可直接使用。

---

## 🌍实时聊天室翻译

整合 Twitch 与 YouTube 聊天室消息。

自动翻译外语消息，让你更轻松地与世界各地的观众交流。

支持头像、徽章、平台图标显示，以及完整 CSS 自定义与 Steam Workshop 主题分享。

## 🎙️ 语音翻译字幕

按住快捷键开始录音。

自动将发言翻译成指定语言并显示在直播画面中，让不同语言的观众也能轻松理解你的内容。

## 🎨 Steam Workshop 主题

下载社区制作的主题，或打造属于自己的 Overlay 风格。

从聊天室到目标条、赞助通知与计时器，都能自由调整外观与动画效果，让你的直播画面不再千篇一律。

支持 Steam Workshop 分享与下载。

## 🎮 常用直播工具一次整合

内置多种直播常用功能：

- 转盘
- 实时投票 / 抽奖
- Twitch 目标条
- 赞助条 / 赞助通知 / 赞助记录板
- 计时器
- 时钟
- 点歌
- 实况效果板（图片 / 视频 / 音效）
- 表情墙
- 精彩标记

无需再安装多个插件或打开多个网页工具。

## ⚡ 专为 OBS 设计

每个功能都提供独立的 OBS 地址，设置完成后复制粘贴即可使用。

聊天室、目标条、计时器、赞助通知等组件均可独立运行，想用什么开什么，不使用的功能不会影响性能。

减少在多个网站、插件与工具之间来回切换，把时间留给直播本身。

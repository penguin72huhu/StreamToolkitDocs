# Twitch 设置

Stream Toolkit 使用 OAuth 授权连结你的 Twitch 账号，不需要手动填入金钥。

## 步骤一：连结 Twitch 账号

1. 开启 Stream Toolkit
2. 点选左下菜单的 **设置**
3. 平台连接中找到 **Twitch**，并点选 **连接账号**
4. 浏览器会自动开启 Twitch 授权页面
5. 登录并点击 **授权**

授权完成后，Stream Toolkit 会自动取得所需的存取金钥。

## 常见问题

**Q：授权后没有反应？**
确认浏览器没有封锁弹出 status 窗口。Stream Toolkit 会开启一個新窗口完成授权。

**Q：需要 Twitch 合作伙伴或联盟主才能用吗？**
不需要。一般账号也可以使用聊天室功能。订阅/Bits 目标需要开通订阅功能。

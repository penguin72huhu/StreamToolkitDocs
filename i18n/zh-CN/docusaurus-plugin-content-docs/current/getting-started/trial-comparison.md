# 试用版 / 正式版比较

Stream Toolkit 提供**免费试用版**与**正式版**。试用版没有时间限制，可以永久使用大部分互动功能，让你充分体验；正式版则解锁金流串接、Steam 工作坊、进阶客制与各项数量限制。

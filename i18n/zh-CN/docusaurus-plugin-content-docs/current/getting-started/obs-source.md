# 如何在 OBS 加入来源

Stream Toolkit 的大部分功能都需要在 OBS 中加入“浏览器源”才能显示在直播画面上。

## 步骤一：复制功能网址

在 Stream Toolkit 中，每个功能页面都会显示对应的本地网址，格式为：

```
http://127.0.0.1:7272/feature-name
```

复制你想加入的功能网址。

## 步骤二：在 OBS 新增浏览器源

1. 在 OBS 的 **来源** 区域，点击下方的 **+** 按钮
2. 选择 **浏览器**
3. 选择 **建立新的**，输入来源名称（例如“赞助条”），点击 **确定**
4. 在 **URL** 栏位贴上步骤一复制的网址
5. 依需求调整 **宽度** 与 **高度**
6. 点击 **确定**

## 需要勾选“使用 OBS 控制音频”的功能以下功能

包含音频输出，必须在浏览器源的设置中勾选 **使用 OBS 控制音频**，声音才能正确出现在直播中：

- **实况效果板** — 视频、音效
- **赞助通知**
- **点歌**

勾选方式：
1. 在 OBS 来源列表中，对该浏览器源点击右键
2. 选择 **属性**
3. 找到 **使用 OBS 控制音频** 并勾选
4. 点击 **确定**

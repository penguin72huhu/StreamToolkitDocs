# 安装與启动

## 系统需求

- Windows 10 / 11 (64-bit)
- 网络连接

## 下载安装

1. 前往 [Steam 页面](https://store.steampowered.com/app/4828880/Stream_Toolkit/) 购买 Stream Toolkit
2. 在 Steam 中点击“安装”，等待下载完成
3. 从 Steam 启动 Stream Toolkit

## 第一次启动

启动后会看到主控制界面，左侧菜单列出所有功能。

Stream Toolkit 的每个功能都有对应的本地网址（例如 `http://127.0.0.1:7272/donate`），在 OBS 中新增“浏览器源”并填入该网址，就能让功能画面出现在直播画面上。

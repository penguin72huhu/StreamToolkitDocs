# Porovnání zkušební a plné verze

Stream Toolkit nabízí **bezplatnou zkušební verzi** a **plnou verzi**. Zkušební verze nemá žádné časové omezení a umožňuje trvalé používání většiny interaktivních funkcí pro plnohodnotný zážitek; plná verze odemyká platební brány, Steam Workshop, pokročilé přizpůsobení a odstraňuje různá kapacitní omezení.

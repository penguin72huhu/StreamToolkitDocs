# Získání DeepL API Key

Klíč DeepL API se používá pro automatický překlad následujících funkcí:
- **Překlad řeči** — Automatický překlad po převodu řeči na text
- **Chat** — Automatický překlad zpráv diváků

## Krok 1: Přihlaste se ke svému účtu DeepL

Přejděte na [DeepL](https://www.deepl.com) a přihlaste se ke svému účtu. Pokud ještě nemáte účet, musíte se nejprve zaregistrovat.

## Krok 2: Přejděte do nastavení Account

Klikněte na **ikonu profilu** v pravém horním rohu a vyberte **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Krok 3: Přepněte na kartu API Keys & limits

Klikněte na kartu **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Krok 4: Vytvořte nový API Key

1. Klikněte na **Create key +**
2. **Name your key**: Zadejte libovolný název (např. `Stream Toolkit`)
3. **Permissions**: Vyberte **All access**
4. Klikněte na **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Krok 5: Zkopírujte a vložte do App

1. Zkopírujte vygenerovaný API Key
2. Vraťte se do Stream Toolkit a vložte jej do odpovídajícího pole **DeepL API Key**

![Paste API Key](./deepl-api-img/step5.png)

## Často kladené otázky

**Q: Má bezplatná zkušební verze DeepL limity použití?**
Ano. Bezplatná zkušební verze poskytuje kvótu 1 000 000 znaků a je omezena na jeden měsíc. Pro další používání vysoce kvalitního překladu si prosím předplaťte placený tarif DeepL.

**Q: Co dělat, když dojde k úniku mého API Key?**
Vraťte se do DeepL Account → API Keys & limits, odstraňte starý Key a vytvořte nový.

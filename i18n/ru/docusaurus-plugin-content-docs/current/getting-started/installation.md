# Установка и запуск

## Системные требования

- Windows 10 / 11 (64-bit)
- Подключение к интернету

## Скачивание и установка

1. Перейдите на [страницу Steam](https://store.steampowered.com/app/4828880/Stream_Toolkit/), чтобы приобрести Stream Toolkit
2. Нажмите «Установить» в Steam и дождитесь завершения загрузки
3. Запустите Stream Toolkit из Steam

## Первый запуск

После запуска вы увидите главный интерфейс управления, в левом меню которого перечислены все функции.

Каждая функция Stream Toolkit имеет соответствующий локальный URL-адрес (например, `http://127.0.0.1:7272/donate`). Добавив «Браузер» (Browser Source) в OBS и введя этот URL, вы сможете вывести экран функции в прямой эфир.

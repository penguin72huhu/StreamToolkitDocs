# Как добавить источники в OBS

Для отображения большинства функций Stream Toolkit в прямом эфире необходимо добавить «Браузер» (Browser Source) в OBS.

## Шаг 1: Скопируйте URL-адрес функции

В Stream Toolkit на странице каждой функции отображается соответствующий локальный URL-адрес в формате:

```
http://127.0.0.1:7272/feature-name
```

Скопируйте URL-адрес функции, которую вы хотите добавить.

## Шаг 2: Добавьте «Браузер» в OBS

1. В разделе **«Источники»** в OBS нажмите кнопку **+** внизу
2. Выберите **«Браузер»**
3. Выберите **«Создать новый»**, введите имя источника (например, «Полоса донатов») и нажмите **«ОК»**
4. Вставьте скопированный на шаге 1 URL-адрес в поле **«URL»**
5. Настройте **«Ширину»** и **«Высоту»** по своему усмотрению
6. Нажмите **«ОК»**

## Ниже приведены функции, требующие включения опции «Управлять аудио с помощью OBS»

Включает вывод аудио; чтобы звук правильно воспроизводился в трансляции, необходимо в настройках источника «Браузер» включить опцию **«Управлять аудио с помощью OBS»**:

- **Media Board** — Видео, Звук
- **Уведомления о донатах**
- **Заказ музыки**

Способ включения:
1. В списке источников OBS нажмите правой кнопкой мыши на этот источник «Браузер»
2. Выберите **«Свойства»**
3. Найдите опцию **«Управлять аудио с помощью OBS»** и включите её
4. Нажмите **«ОК»**

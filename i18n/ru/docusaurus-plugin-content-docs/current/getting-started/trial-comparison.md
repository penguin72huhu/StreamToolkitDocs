# Сравнение пробной и полной версий

Stream Toolkit предлагает **бесплатную пробную версию** и **полную версию**. Пробная версия не имеет ограничений по времени и позволяет постоянно использовать большинство интерактивных функций для полноценного ознакомления; полная версия открывает доступ к интеграции платежных систем, Мастерской Steam, продвинутой кастомизации и снимает различные количественные ограничения.

:::info Об обновлении версии
Пробная и полная версии являются двумя отдельными приложениями в Steam. После покупки полной версии её необходимо установить отдельно, при этом настройки и записи между ними не переносятся.
:::

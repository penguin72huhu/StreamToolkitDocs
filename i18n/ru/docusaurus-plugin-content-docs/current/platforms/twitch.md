# Twitch Настройки

Stream Toolkit использует авторизацию OAuth для подключения вашего аккаунта Twitch, поэтому вводить ключ вручную не требуется.

## Шаг 1: Подключить аккаунт Twitch

1. Откройте Stream Toolkit
2. Нажмите **Настройки** в левом нижнем меню
3. Найдите **Twitch** в подключениях платформ и нажмите **Подключить аккаунт**
4. Браузер автоматически откроет страницу авторизации Twitch
5. Войдите и нажмите **Авторизовать**

После завершения авторизации Stream Toolkit автоматически получит необходимый ключ доступа.

## Часто задаваемые вопросы

**Q: Нет реакции после авторизации?**
Убедитесь, что браузер не блокирует всплывающие окна. Stream Toolkit откроет новое окно для завершения авторизации.

**Q: Нужно ли быть партнером или компаньоном Twitch, чтобы пользоваться этим?**
Нет. Обычные аккаунты также могут использовать функции чата. Для целей по подпискам/Bits необходимо, чтобы на канале была включена возможность подписки.

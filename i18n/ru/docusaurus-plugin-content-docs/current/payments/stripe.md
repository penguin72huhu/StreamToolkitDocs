# Настройка Stripe

Stream Toolkit получает уведомления о платежах Stripe через Webhook. Настройка состоит из двух частей: получение URL-адреса Webhook из app и выполнение интеграции в панели управления Stripe.

## Шаг 1: Получите URL-адрес Webhook в Stream Toolkit

1. Откройте Stream Toolkit
2. Нажмите **Настройки** в левом нижнем меню → **Интеграция платформ донатов** → **Stripe** (Нажмите, чтобы развернуть)
3. Вы увидите **Webhook URL**, формат указан ниже:
   ```
   https://<worker>/stripe/webhook/<your userId>
   ```
4. Нажмите кнопку **Копировать** и сохраните этот URL-адрес для дальнейшего использования

![Get webhook URL](./stripe-img/step1.png)

## Шаг 2: Добавьте Webhook в панели управления Stripe

1. Перейдите в [Stripe Dashboard](https://dashboard.stripe.com) и войдите в свой аккаунт
2. Нажмите **Разработчикам** → **Webhook** в левом нижнем углу

![Stripe webhook list page](./stripe-img/step2-1.png)

3. Нажмите **Добавить конечную точку**

![Add destination](./stripe-img/step2-2.png)

4. Заполните следующую информацию:
   - **События**: Найдите и отметьте `checkout.session.completed` (нужно только это одно)

   ![Select events](./stripe-img/step2-3.png)

   - **Тип конечной точки**: Выберите **Конечная точка Webhook**

   ![Select destination type](./stripe-img/step2-4.png)

   - **Имя конечной точки**: Укажите любое имя (например, `Stream Toolkit`)
   - **URL конечной точки**: Вставьте URL-адрес Webhook, скопированный на шаге 1

   ![Enter name and URL](./stripe-img/step2-5.png)

5. Нажмите **Добавить конечную точку**

## Шаг 3: Введите секрет подписи

1. После создания Webhook на странице отобразится **секрет подписи** в формате `whsec_...`
2. Скопируйте этот секрет
3. Вернитесь в раздел настроек Stripe в Stream Toolkit
4. Вставьте секрет в поле **Секрет подписи Webhook**
5. Нажмите **Сохранить**

Если статус подключения изменился на зеленый, настройка выполнена успешно.

![Paste signing secret](./stripe-img/step3.png)

## Готово

После завершения настройки, когда зрители будут совершать оплату через вашу Stripe **Payment Link**, Stream Toolkit будет мгновенно получать уведомления и отображать донат.

## Часто задаваемые вопросы

**Q: Где можно создать Payment Link?**
Перейдите в Stripe Dashboard → **Payment Links** → **Создать Payment Link**, установите сумму и поделитесь ссылкой со зрителями.

**Q: Статус подключения не стал зеленым?**
Убедитесь, что Секрет подписи Webhook правильно вставлен и сохранен, а URL-адрес конечной точки в панели Stripe полностью совпадает с тем, что отображается в app.

# Hämta DeepL API Key

DeepL API-nyckeln används för automatisk översättning av följande funktioner:
- **Rösttolkning** — Automatisk översättning efter konvertering av tal till text
- **Chatt** — Automatisk översättning av tittarmeddelanden

## Steg 1: Logga in på ditt DeepL-konto

Gå till [DeepL](https://www.deepl.com) och logga in på ditt konto. Om du inte har ett konto ännu måste du registrera dig först.

## Steg 2: Gå till Account-inställningar

Klicka på **profilikonen** i det övre högra hörnet och välj **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Steg 3: Växla till fliken API Keys & limits

Klicka på fliken **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Steg 4: Skapa en ny API Key

1. Klicka på **Create key +**
2. **Name your key**: Ange valfritt namn (t.ex. `Stream Toolkit`)
3. **Permissions**: Välj **All access**
4. Klicka på **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Steg 5: Kopiera och klistra in i Appen

1. Kopiera den genererade API Key:n
2. Gå tillbaka till Stream Toolkit och klistra in i motsvarande **DeepL API Key**-fält

![Paste API Key](./deepl-api-img/step5.png)

## Vanliga frågor

**Q: Har DeepL:s gratis testversion användningsgränser?**
Ja. Den gratis testversionen ger en kvot på 1 000 000 tecken och är begränsad till en månad. För fortsatt användning av översättningar av hög kvalitet, vänligen prenumerera på en betald DeepL-plan.

**Q: Vad ska jag göra om min API Key läcker?**
Gå till DeepL Account → API Keys & limits, ta bort den gamla Key:n och skapa en ny.

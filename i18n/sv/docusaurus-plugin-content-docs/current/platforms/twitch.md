# Twitch Inställningar

Stream Toolkit använder OAuth-auktorisering för att ansluta ditt Twitch-konto, så det finns inget behov av att ange en nyckel manuellt.

## Steg 1: Anslut Twitch-konto

1. Öppna Stream Toolkit
2. Klicka på **Inställningar** i den nedre vänstra menyn
3. Hitta **Twitch** under Plattformanslutningar och klicka på **Anslut konto**
4. Webbläsaren öppnar automatiskt Twitch-auktoriseringssidan
5. Logga in och klicka på **Auktorisera**

När auktoriseringen är klar kommer Stream Toolkit automatiskt att hämta den åtkomstnyckel जो krävs.

## Vanliga frågor

**Q: Ingen reaktion efter auktorisering?**
Se till att din webbläsare inte blockerar pop-up-fönster. Stream Toolkit öppnar ett nytt fönster för att slutföra auktoriseringen.

**Q: Måste jag vara Twitch-partner eller Affiliate för att använda det?**
Nej. Normala konton kan också använda chattfunktionerna. Prenumerations-/Bits-mål kräver att prenumerationsfunktioner är aktiverade.

# Installation och start

## Systemkrav

- Windows 10 / 11 (64-bit)
- Internetanslutning

## Nedladdning och installation

1. Gå till [Steam-sidan](https://store.steampowered.com/app/4828880/Stream_Toolkit/) för att köpa Stream Toolkit
2. Klicka på "Installera" i Steam och vänta tills nedladdningen är klar
3. Starta Stream Toolkit från Steam

## Första starten

Efter start kommer du att se huvudkontrollsgränssnittet, där vänstermenyn listar alla funktioner.

Varje funktion i Stream Toolkit har en motsvarande lokal URL (t.ex. `http://127.0.0.1:7272/donate`). Genom att lägga till en "Webbläsarkälla" i OBS och ange den URL-en kan du visa funktionens skärm i livesändningen.

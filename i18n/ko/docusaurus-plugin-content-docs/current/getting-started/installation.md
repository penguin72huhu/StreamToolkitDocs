# 설치 및 실행

## 시스템 요구 사항

- Windows 10 / 11 (64-bit)
- 인터넷 연결

## 다운로드 및 설치

1. [Steam 페이지](https://store.steampowered.com/app/4828880/Stream_Toolkit/)로 이동하여 Stream Toolkit을 구매합니다
2. Steam에서 "설치"를 클릭하고 다운로드가 완료될 때까지 기다립니다
3. Steam에서 Stream Toolkit을 실행합니다

## 첫 실행

실행 후 메인 제어 인터페이스가 표시되며, Diagnostic 왼쪽 메뉴에 모든 기능이 나열됩니다.

Stream Toolkit의 각 기능에는 해당하는 로컬 URL(예: `http://127.0.0.1:7272/donate`)이 있습니다. OBS에서 "브라우저 소스"를 추가하고 해당 URL을 입력하면 라이브 방송 화면에 기능 화면을 표시할 수 있습니다.

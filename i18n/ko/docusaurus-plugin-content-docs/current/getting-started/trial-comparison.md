# 체험판 / 정식판 비교

Stream Toolkit은 **무료 체험판**과 **정식판**을 제공합니다. 체험판은 시간 제한이 없어 대부분의 인터랙티브 기능을 영구적으로 사용하며 충분히 체험할 수 있습니다. 정식판은 결제 연동, Steam 창작마당, 고급 맞춤 설정 및 각종 수량 제한 해제를 제공합니다.

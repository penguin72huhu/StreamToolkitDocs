# OBS에서 소스 추가하는 방법

Stream Toolkit의 대부분의 기능은 라이브 방송 화면에 표시하기 위해 OBS에서 "브라우저 소스"를 추가해야 합니다.

## 1단계: 기능 URL 복사

Stream Toolkit에서는 각 기능 페이지마다 해당하는 로컬 URL이 다음 형식으로 표시됩니다.

```
http://127.0.0.1:7272/feature-name
```

추가하려는 기능의 URL을 복사합니다.

## 2단계: OBS에서 브라우저 소스 추가

1. OBS의 **소스** 섹션에서 하단의 **+** 버튼을 클릭합니다
2. **브라우저**를 선택합니다
3. **새로 만들기**를 선택하고 소스 이름(예: "후원 바")을 입력한 후 **확인**을 클릭합니다
4. **URL** 필드에 1단계에서 복사한 URL을 붙여넣습니다
5. 필요에 따라 **너비**와 **높이**를 조절합니다
6. **확인**을 클릭합니다

## 아래 기능들은 "OBS를 통해 오디오 제어"를 체크해야 합니다

오디오 출력을 포함하므로, 방송에 소리가 올바르게 나오려면 브라우저 소스 설정에서 **OBS를 통해 오디오 제어**를 체크해야 합니다:

- **미디어 보드** — 영상, 사운드
- **후원 알림**
- **노래 신청**

체크 방법:
1. OBS 소스 목록에서 해당 브라우저 소스를 마우스 오른쪽 버튼으로 클릭합니다
2. **속성**을 선택합니다
3. **OBS를 통해 오디오 제어**를 찾아 체크합니다
4. **확인**을 클릭합니다

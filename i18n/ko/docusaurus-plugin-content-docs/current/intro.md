# Stream Toolkit 소개

왼쪽 메뉴는 기능별로 분류되어 있습니다. 초보자는 [빠른 시작](./getting-started/installation)부터 읽는 것을 권장합니다.

---

Stream Toolkit은 스트리머를 위한 올인원 스트리밍 툴킷으로, 여러 웹사이트를 번갈아 사용하는 번거로움을 줄여줍니다.

- **실시간 번역**: 채팅과 음성 번역 기능으로 전 세계 시청자와 쉽게 소통할 수 있습니다.
- **다양한 기능 통합**: 투표, 추첨, 목표 바, 후원 알림, 미디어 보드를 하나의 프로그램에 통합했습니다.
- **커스터마이징**: CSS 커스터마이징과 Steam Workshop 테마를 지원하여 자유롭게 스타일을 변경할 수 있습니다.
- **간편한 설정**: Overlay URL을 한 번의 클릭으로 생성하여 OBS에 붙여넣기만 하면 바로 사용할 수 있습니다.

---

## 🌍 실시간 채팅 번역

Twitch와 YouTube 채팅 메시지를 통합합니다.

외국어 메시지를 자동으로 번역하여 전 세계 시청자와 더욱 쉽게 소통할 수 있습니다.

아바타, 배지, 플랫폼 아이콘 표시를 지원하며, CSS 커스터마이징과 Steam Workshop 테마 공유도 지원합니다.

## 🎙️ 번역 자막

단축키를 눌러 녹음을 시작합니다.

발언 내용을 지정한 언어로 번역하여 방송 화면에 표시하므로, 다른 언어를 사용하는 시청자도 쉽게 내용을 이해할 수 있습니다.

## 🎨 Steam Workshop 테마

커뮤니티가 제작한 테마를 다운로드하거나 자신만의 Overlay 스타일을 만들 수 있습니다.

채팅, 목표 바, 후원 알림, 타이머의 외형과 애니메이션을 자유롭게 조정하여 자신만의 방송 화면을 만들 수 있습니다.

Steam Workshop 공유 및 다운로드를 지원합니다.

## 🎮 다양한 스트리밍 도구 통합

다양한 스트리밍 기능을 기본 제공합니다:

- 룰렛
- 실시간 투표 / 추첨
- Twitch 목표 바
- 후원 바 / 후원 알림 / 후원 기록판
- 타이머
- 시계
- 노래 신청
- 미디어 보드 (이미지 / 영상 / 사운드)
- 이모트 월
- 하이라이트 마커

여러 플러그인이나 웹 도구를 설치할 필요가 없습니다.

## ⚡ OBS를 위한 설계

모든 기능은 독립적인 OBS URL을 제공합니다.설정 후 복사하여 붙여넣기만 하면 바로 사용할 수 있습니다.

채팅, 목표 바, 타이머, 후원 알림 등은 각각 독립적으로 동작하며 사용하지 않는 기능은 성능에 영향을 주지 않습니다.

여러 웹사이트와 플러그인, 도구를 오가는 시간을 줄이고 스트리밍 자체에 집중할 수 있습니다.

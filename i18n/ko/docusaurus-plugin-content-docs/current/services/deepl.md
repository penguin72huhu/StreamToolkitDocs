# DeepL API Key 획득

DeepL API Key는 다음 기능의 자동 번역에 사용됩니다:
- **음성 번역** — 음성을 텍스트로 변환 후 자동 번역
- **채팅방** — 시청자 메시지 자동 번역

## 1단계: DeepL 계정 로그인

[DeepL](https://www.deepl.com)에 접속하여 계정에 로그인하세요. 아직 계정이 없는 경우, 먼저 회원가입이 필요합니다.

## 2단계: Account 설정 진입하기

우측 상단의 **프로필 아이콘**을 클릭하고 **Account**를 선택하세요.

![Open Account settings](./deepl-api-img/step2.png)

## 3단계: API Keys & limits 탭으로 전환하기

**API Keys & limits** 탭을 클릭하세요.

![API Keys & limits tab](./deepl-api-img/step3.png)

## 4단계: 새 API Key 생성하기

1. **Create key +**를 클릭하세요
2. **Name your key**: 원하는 이름을 입력하세요 (예: `Stream Toolkit`)
3. **Permissions**: **All access**를 선택하세요
4. **Create Key**를 클릭하세요

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## 5단계: 복사하여 App에 붙여넣기

1. 생성된 API Key를 복사하세요
2. Stream Toolkit으로 돌아가 해당 **DeepL API Key** 필드에 붙여넣으세요

![Paste API Key](./deepl-api-img/step5.png)

## 자주 묻는 질문

**Q: DeepL 무료 체험판에 사용량 제한이 있나요?**
네. 무료 체험판은 1,000,000 자의 할당량을 제공하며 한 달 이내에만 사용할 수 있습니다. 고품질 번역을 계속 사용하려면 DeepL 유료 요금제를 구독하세요.

**Q: API Key가 유출되면 어떻게 해야 하나요?**
DeepL Account → API Keys & limits로 돌아가 기존 Key를 삭제하고 새로 만드세요.

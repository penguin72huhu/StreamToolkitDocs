# Twitch 설정

Stream Toolkit은 OAuth 인증을 사용하여 Twitch 계정을 연결하므로 키를 수동으로 입력할 필요가 없습니다.

## 1단계: Twitch 계정 연결

1. Stream Toolkit을 엽니다
2. 왼쪽 아래 메뉴에서 **설정**을 클릭합니다
3. 플랫폼 연동에서 **Twitch**를 찾아 **계정 연결**을 클릭합니다
4. 브라우저에서 Twitch 인증 페이지가 자동으로 열립니다
5. 로그인 후 **승인**을 클릭합니다

승인이 완료되면 Stream Toolkit이 필요한 액세스 키를 자동으로 가져옵니다.

## 자주 묻는 질문

**Q: 승인 후 아무런 반응이 없나요?**
브라우저가 팝업 창을 차단하지 않았는지 확인하세요. Stream Toolkit이 새 창을 열어 승인을 완료합니다.

**Q: Twitch 파트너나 제휴 회원이어야 사용할 수 있나요?**
아닙니다. 일반 계정도 채팅창 기능을 사용할 수 있습니다. 구독/Bits 목표는 구독 기능이 활성화되어 있어야 합니다.

# Cài đặt và Khởi chạy

## Yêu cầu hệ thống

- Windows 10 / 11 (64-bit)
- Kết nối internet

## Tải xuống và cài đặt

1. Truy cập [Trang Steam](https://store.steampowered.com/app/4828880/Stream_Toolkit/) để mua Stream Toolkit
2. Nhấp vào "Cài đặt" trong Steam và chờ quá trình tải xuống hoàn tất
3. Khởi chạy Stream Toolkit từ Steam

## Khởi chạy lần đầu

Sau khi khởi chạy, bạn sẽ thấy giao diện điều khiển chính, với menu bên trái liệt kê tất cả các tính năng.

Mỗi tính năng của Stream Toolkit đều có một URL cục bộ tương ứng (ví dụ: `http://127.0.0.1:7272/donate`). Bằng cách thêm "Nguồn trình duyệt" (Browser Source) trong OBS và điền URL đó, bạn có thể hiển thị màn hình tính năng trên luồng trực tiếp.

# Nhận DeepL API Key

DeepL API Key được sử dụng để dịch tự động các tính năng sau:
- **Dịch giọng nói** — Dịch tự động sau khi chuyển đổi giọng nói thành văn bản
- **Phòng chat** — Dịch tự động tin nhắn của người xem

## Bước 1: Đăng nhập vào tài khoản DeepL

Truy cập [DeepL](https://www.deepl.com) và đăng nhập vào tài khoản của bạn. Nếu bạn chưa có tài khoản, bạn cần đăng ký trước.

## Bước 2: Truy cập cài đặt Account

Nhấp vào **biểu tượng hồ sơ** ở góc trên bên phải và chọn **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Bước 3: Chuyển sang tab API Keys & limits

Nhấp vào tab **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Bước 4: Tạo API Key mới

1. Nhấp vào **Create key +**
2. **Name your key**: Nhập tên bất kỳ (ví dụ: `Stream Toolkit`)
3. **Permissions**: Chọn **All access**
4. Nhấp vào **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Bước 5: Sao chép và dán vào App

1. Sao chép API Key đã tạo
2. Quay lại Stream Toolkit và dán vào trường **DeepL API Key** tương ứng

![Paste API Key](./deepl-api-img/step5.png)

## Câu hỏi thường gặp

**Q: Phiên bản dùng thử miễn phí của DeepL có giới hạn sử dụng không?**
Có. Phiên bản dùng thử miễn phí cung cấp hạn mức 1.000.000 ký tự và chỉ giới hạn trong một tháng. Để tiếp tục sử dụng dịch thuật chất lượng cao, vui lòng đăng ký gói trả phí của DeepL.

**Q: Tôi phải làm gì nếu API Key bị rò rỉ?**
Quay lại DeepL Account → API Keys & limits, xóa Key cũ và tạo một Key mới.

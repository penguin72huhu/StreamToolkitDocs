# Instalação e inicialização

## Requisitos do sistema

- Windows 10 / 11 (64-bit)
- Conexão com a Internet

## Download e instalação

1. Acesse a [página do Steam](https://store.steampowered.com/app/4828880/Stream_Toolkit/) para comprar o Stream Toolkit
2. Clique em "Instalar" no Steam e aguarde o download ser concluído
3. Inicie o Stream Toolkit a partir do Steam

## Primeira inicialização

Após a inicialização, você verá a interface de controle principal, com o menu esquerdo listando todos os recursos.

Cada recurso do Stream Toolkit possui uma URL local correspondente (por exemplo, `http://127.0.0.1:7272/donate`). Ao adicionar uma "Fonte de Navegador" no OBS e inserir essa URL, você poderá exibir a tela do recurso na sua transmissão ao vivo.

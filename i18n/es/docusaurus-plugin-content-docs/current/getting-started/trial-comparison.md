# Comparación entre versión de prueba y versión completa

Stream Toolkit ofrece una **versión de prueba gratuita** y una **versión completa**. La versión de prueba no tiene límite de tiempo y permite el uso permanente de la mayoría de las funciones interactivas para una experiencia completa; la versión completa desbloquea pasarelas de pago, Steam Workshop, personalización avanzada y elimina varios límites de cantidad.

:::info Acerca de la actualización
La versión de prueba y la versión completa son dos aplicaciones independientes en Steam. Después de comprar la versión completa, debe instalarse por separado, y los ajustes y registros no se comparten entre sí.
:::

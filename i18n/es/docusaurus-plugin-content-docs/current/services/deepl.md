# Obtención de la clave API de DeepL

La clave API de DeepL se utiliza para la traducción automática de las siguientes funciones:
- **Traducción de voz** — Traducción automática tras convertir voz a texto
- **Chat** — Traducción automática de mensajes de espectadores

## Paso 1: Iniciar sesión en tu cuenta de DeepL

Ve a [DeepL](https://www.deepl.com) e inicia sesión en tu cuenta. Si aún no tienes una cuenta, primero debes registrarte.

## Paso 2: Ir a la configuración de Account

Haz clic en el **icono de perfil** en la esquina superior derecha y selecciona **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Paso 3: Cambiar a la pestaña API Keys & limits

Haz clic en la pestaña **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Paso 4: Crear una nueva API Key

1. Haz clic en **Create key +**
2. **Name your key**: Introduce cualquier nombre (por ejemplo, `Stream Toolkit`)
3. **Permissions**: Selecciona **All access**
4. Haz clic en **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Paso 5: Copiar y pegar en la App

1. Copia la API Key generada
2. Vuelve a Stream Toolkit y pégalo en el campo **DeepL API Key** correspondiente

![Paste API Key](./deepl-api-img/step5.png)

## Preguntas frecuentes

**Q: ¿Tiene límites de uso la versión de prueba gratuita de DeepL?**
Sí. La versión de prueba gratuita ofrece un límite de 1,000,000 de caracteres y está limitada a un mes. Para seguir utilizando traducciones de alta calidad, suscríbete a un plan de pago de DeepL.

**Q: ¿Qué hago si se filtra mi API Key?**
Vuelve a DeepL Account → API Keys & limits, elimina la Key antigua y crea una nueva.

# Stream Toolkit Introducción

El menú izquierdo está clasificado por funciones. Se recomienda a los principiantes comenzar a leer desde [Inicio rápido](./getting-started/installation).

---

Stream Toolkit es una caja de herramientas de streaming todo en uno diseñada para creadores de contenido, evitando tener que cambiar constantemente entre diferentes sitios web.

- **Traducción en tiempo real**: la traducción integrada de chat y voz facilita la comunicación con espectadores de todo el mundo.
- **Funciones integradas**: encuestas, sorteos, barras de objetivos, alertas de donaciones y Media Board se integran en una sola aplicación.
- **Personalización**: compatible con personalización CSS y temas de Steam Workshop.
- **Configuración sencilla**: genera una URL Overlay con un solo clic y pégala directamente en OBS.

---

## 🌍 Traducción de chat en tiempo real

Integra mensajes de Twitch y YouTube en un solo lugar.

Traduce automáticamente mensajes en otros idiomas y comunícate más fácilmente con espectadores de todo el mundo.

Compatible con avatares, insignias, iconos de plataforma, personalización CSS y temas de Steam Workshop.

## 🎙️ Subtítulos traducidos

Mantén presionada una tecla rápida para comenzar a grabar.

Traduce automáticamente tu voz al idioma seleccionado y muestra los subtítulos durante la transmisión.

## 🎨 Temas de Steam Workshop

Descarga temas creados por la comunidad o crea tu propio estilo Overlay.

Personaliza libremente la apariencia y las animaciones del chat, las barras de objetivos, las alertas de donaciones y los temporizadores.

Compatible con compartir y descargar desde Steam Workshop.

## 🎮 Todas tus herramientas de streaming en un solo lugar

Funciones incluidas:

- Ruleta
- Encuestas en vivo / Sorteos
- Barra de objetivos de Twitch
- Barra de donaciones / Alertas de donaciones / Panel de donaciones
- Temporizador
- Reloj
- Solicitudes de canciones
- Media Board (Imágenes / Vídeos / Sonidos)
- Muro de emotes
- Marcador de momentos destacados

No necesitas instalar múltiples complementos ni utilizar varias herramientas web.

## ⚡ Diseñado para OBS

Cada función incluye su propia URL de OBS.Simplemente copia y pega para comenzar.

Los chats, barras de objetivos, temporizadores y alertas funcionan de forma independiente.

Dedica menos tiempo a administrar herramientas y más tiempo a transmitir.

# Twitch Einstellungen

Stream Toolkit verwendet die OAuth-Autorisierung, um Ihr Twitch-Konto zu verbinden, sodass kein Schlüssel manuell eingegeben werden muss.

## Schritt 1: Twitch-Konto verbinden

1. Öffnen Sie Stream Toolkit
2. Klicken Sie im linken unteren Menü auf **Einstellungen**
3. Suchen Sie **Twitch** unter Plattformverbindungen und klicken Sie auf **Konto verbinden**
4. Der Browser öffnet automatisch die Twitch-Autorisierungsseite
5. Melden Sie sich an und klicken Sie auf **Autorisieren**

Nach erfolgreicher Autorisierung ruft Stream Toolkit automatisch den erforderlichen Zugriffsschlüssel ab.

## Häufig gestellte Fragen (FAQ)

**Q: Keine Reaktion nach der Autorisierung?**
Stellen Sie sicher, dass Ihr Browser keine Pop-up-Fenster blockiert. Stream Toolkit öffnet ein neues Fenster, um die Autorisierung abzuschließen.

**Q: Muss ich ein Twitch-Partner oder -Affiliate sein, um es zu nutzen?**
Nein. Normale Konten können die Chat-Funktionen ebenfalls nutzen. Abonnement-/Bits-Ziele erfordern die Freischaltung der Abonnement-Funktionen.

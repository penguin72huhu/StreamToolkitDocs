# Installation und Start

## Systemanforderungen

- Windows 10 / 11 (64-bit)
- Internetverbindung

## Download und Installation

1. Gehe zur [Steam-Seite](https://store.steampowered.com/app/4828880/Stream_Toolkit/), um Stream Toolkit zu kaufen
2. Klicke in Steam auf „Installieren“ und warte, bis der Download abgeschlossen ist
3. Starte Stream Toolkit über Steam

## Erster Start

Nach dem Start sehen Sie die Hauptbedienoberfläche, wobei das linke Menü alle Funktionen auflistet.

Jede Funktion von Stream Toolkit verfügt über eine entsprechende lokale URL (z. B. `http://127.0.0.1:7272/donate`). Wenn Sie in OBS eine „Browserquelle“ hinzufügen und diese URL eingeben, wird der Funktionsbildschirm im Live-Stream angezeigt.

# Vergleich zwischen Testversion und Vollversion

Stream Toolkit bietet eine **kostenlose Testversion** und eine **Vollversion**. Die Testversion hat kein Zeitlimit und ermöglicht die dauerhafte Nutzung der meisten interaktiven Funktionen für ein vollständiges Erlebnis; die Vollversion schaltet Zahlungsanbindungen, den Steam Workshop, erweiterte Anpassungen und verschiedene Mengeneinschränkungen frei.

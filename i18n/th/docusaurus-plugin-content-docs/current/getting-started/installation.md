# การติดตั้งและการเปิดใช้งาน

## ความต้องการของระบบ

- Windows 10 / 11 (64-bit)
- การเชื่อมต่ออินเทอร์เน็ต

## ดาวน์โหลดและการติดตั้ง

1. ไปที่ [หน้า Steam](https://store.steampowered.com/app/4828880/Stream_Toolkit/) เพื่อซื้อ Stream Toolkit
2. คลิก "ติดตั้ง" ใน Steam และรอให้การดาวน์โหลดเสร็จสิ้น
3. เปิดใช้งาน Stream Toolkit จาก Steam

## การเปิดใช้งานครั้งแรก

بعدเปิดใช้งาน คุณจะเห็นอินเทอร์เฟซควบคุมหลัก โดยเมนูด้านซ้ายจะแสดงฟังก์ชันทั้งหมด

แต่ละฟังก์ชันของ Stream Toolkit จะมี URL ท้องถิ่นที่สอดคล้องกัน (เช่น `http://127.0.0.1:7272/donate`) โดยการเพิ่ม "เบราว์เซอร์" (Browser Source) ใน OBS แล้วใส่ URL นั้น จะสามารถทำให้หน้าจอของฟังก์ชันแสดงบนหน้าจอไลฟ์สตรีมได้

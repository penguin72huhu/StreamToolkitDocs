# วิธีเพิ่มซอร์สใน OBS

ฟังก์ชันส่วนใหญ่ของ Stream Toolkit จำเป็นต้องเพิ่ม "เบราว์เซอร์" (Browser Source) ใน OBS เพื่อให้แสดงบนหน้าจอไลฟ์สตรีมได้

## ขั้นตอนที่ 1: คัดลอก URL ของฟังก์ชัน

ใน Stream Toolkit แต่ละหน้าฟังก์ชันจะแสดง URL ท้องถิ่นที่สอดคล้องกันในรูปแบบ:

```
http://127.0.0.1:7272/feature-name
```

คัดลอก URL ของฟังก์ชันที่คุณต้องการเพิ่ม

## ขั้นตอนที่ 2: เพิ่มซอร์สเบราว์เซอร์ใน OBS

1. ในส่วน **ซอร์ส** (Sources) ของ OBS ให้คลิกปุ่ม **+** ที่ด้านล่าง
2. เลือก **เบราว์เซอร์**
3. เลือก **สร้างใหม่** ใส่ชื่อซอร์ส (เช่น "แถบเป้าหมายการสนับสนุน") แล้วคลิก **ตกลง**
4. วาง URL ที่คัดลอกมาจากขั้นตอนที่ 1 ในช่อง **URL**
5. ปรับ **ความกว้าง** และ **ความสูง** ตามต้องการ
6. คลิก **ตกลง**

## ฟังก์ชันต่อไปนี้จำเป็นต้องติ๊กเลือก "ควบคุมเสียงผ่าน OBS"

มีเอาต์พุตเสียง จำเป็นต้องติ๊กเลือก **ควบคุมเสียงผ่าน OBS** ในการตั้งค่าซอร์สเบราว์เซอร์ เพื่อให้เสียงแสดงในการไลฟ์สตรีมได้อย่างถูกต้อง:

- **Media Board** — วิดีโอ, เสียง
- **การแจ้งเตือนการสนับสนุน**
- **ขอเพลง**

วิธีการติ๊กเลือก:
1. ในรายการซอร์สของ OBS ให้คลิกขวาที่ซอร์สเบราว์เซอร์นั้น
2. เลือก **คุณสมบัติ**
3. ค้นหา **ควบคุมเสียงผ่าน OBS** แล้วติ๊กเลือก
4. คลิก **ตกลง**

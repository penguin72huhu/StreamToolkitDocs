# Stream Toolkit แนะนํา

เมนูด้านซ้ายแบ่งตามฟังก์ชันการใช้งาน แนะนำสำหรับผู้เริ่มต้นให้เริ่มอ่านจาก [เริ่มต้นใช้งานด่วน](./getting-started/installation)

---

Stream Toolkit คือชุดเครื่องมือสตรีมมิงแบบครบวงจรสำหรับสตรีมเมอร์ ช่วยให้คุณไม่ต้องสลับใช้งานหลายเว็บไซต์และเครื่องมือต่าง ๆ อยู่ตลอดเวลา

- **การแปลแบบเรียลไทม์**: ระบบแปลแชตและเสียงในตัวช่วยให้คุณสื่อสารกับผู้ชมทั่วโลกได้อย่างง่ายดาย
- **รวมทุกอย่างไว้ในที่เดียว**: โพล การจับรางวัล แถบเป้าหมาย การแจ้งเตือนการสนับสนุน และ Media Board รวมอยู่ในโปรแกรมเดียว
- **การปรับแต่ง**: รองรับการปรับแต่งด้วย CSS และธีมจาก Steam Workshop ให้คุณเปลี่ยนสไตล์ได้อย่างอิสระ
- **เริ่มต้นได้อย่างรวดเร็ว**: สร้าง URL Overlay ได้ในคลิกเดียว และนำไปใช้งานกับ OBS ได้ทันที

---

## 🌍 การแปลแชตแบบเรียลไทม์

รวมข้อความจาก Twitch และ YouTube ไว้ในที่เดียว

แปลข้อความภาษาต่างประเทศโดยอัตโนมัติ ช่วยให้คุณสื่อสารกับผู้ชมจากทั่วโลกได้สะดวกยิ่งขึ้น

รองรับรูปโปรไฟล์ ป้าย ไอคอนแพลตฟอร์ม การปรับแต่ง CSS แบบเต็มรูปแบบ และการแชร์ธีมผ่าน Steam Workshop

## 🎙️ ซับไตเติลแปลภาษา

กดปุ่มลัดค้างไว้เพื่อเริ่มบันทึกเสียง

ระบบจะแปลคำพูดของคุณเป็นภาษาที่กำหนดและแสดงบนหน้าจอระหว่างการถ่ายทอดสด ทำให้ผู้ชมที่ใช้ภาษาแตกต่างกันสามารถเข้าใจเนื้อหาได้ง่ายขึ้น

## 🎨 ธีม Steam Workshop

ดาวน์โหลดธีมที่สร้างโดยชุมชน หรือสร้างสไตล์ Overlay ของคุณเอง

ตั้งแต่แชต แถบเป้าหมาย การแจ้งเตือนการสนับสนุน ไปจนถึงตัวจับเวลา คุณสามารถปรับแต่งรูปลักษณ์และแอนิเมชันได้อย่างอิสระ เพื่อให้หน้าสตรีมของคุณมีเอกลักษณ์ไม่เหมือนใคร

รองรับการแชร์และดาวน์โหลดผ่าน Steam Workshop

## 🎮 รวมเครื่องมือสตรีมยอดนิยมไว้ในที่เดียว

ฟังก์ชันที่มี:

- วงล้อสุ่ม
- โพลสด / การจับรางวัล
- แถบเป้าหมาย Twitch
- แถบเป้าหมายการสนับสนุน / การแจ้งเตือนการสนับสนุน / กระดานการสนับสนุน
- ตัวจับเวลา
- นาฬิกา
- ขอเพลง
- Media Board (รูปภาพ / วิดีโอ / เสียง)
- กำแพงอิโมจิ
- ตัวทำเครื่องหมายไฮไลต์

ไม่จำเป็นต้องติดตั้งปลั๊กอินหลายตัวหรือเปิดใช้งานเครื่องมือเว็บหลายเว็บไซต์อีกต่อไป

## ⚡ ออกแบบมาเพื่อ OBS

แต่ละฟังก์ชันมี URL สำหรับ OBS แยกต่างหาก เพียงคัดลอกและวางก็สามารถใช้งานได้ทันที

แชต แถบเป้าหมาย ตัวจับเวลา และการแจ้งเตือนการสนับสนุนสามารถทำงานแยกจากกันได้ เปิดใช้เฉพาะฟังก์ชันที่ต้องการโดยไม่ส่งผลต่อประสิทธิภาพ

ลดเวลาที่ต้องสลับไปมาระหว่างเว็บไซต์ ปลั๊กอิน และเครื่องมือต่าง ๆ เพื่อให้คุณมีสมาธิกับการสตรีมมากยิ่งขึ้น

# การตั้งค่า ECPay

บทเรียนนี้อธิบายวิธีรับ **HashKey** และ **HashIV** จาก ECPay และกรอกลงใน Stream Toolkit

## ขั้นตอนที่ 1: ลงชื่อเข้าใช้หลังบ้านร้านค้าของ ECPay

1. ไปที่ [เว็บไซต์ทางการของ ECPay](https://www.ecpay.com.tw/)
2. คลิกที่ **เข้าสู่ระบบผู้ขาย** → **พื้นที่สำหรับร้านค้า** ที่มุมขวาบน

## ขั้นตอนที่ 2: ไปที่ การตั้งค่าการเชื่อมต่อระบบ

1. คลิกที่ **การตั้งค่าระบบ** ในเมนูด้านซ้าย
2. เลือก **การตั้งค่าการเชื่อมต่อระบบ**

   ![System interface setup in system settings menu](./ecpay-img/step2-2.png)

3. ค้นหา **Hash Key การเชื่อมต่อ** และ **Hash IV การเชื่อมต่อ**

   ![HashKey and HashIV](./ecpay-img/step2-3.png)

## ขั้นตอนที่ 3: กรอกข้อมูลลงใน Stream Toolkit

1. เปิด Stream Toolkit
2. คลิกที่ **ตั้งค่า** ในเมนูด้านซ้ายล่าง
3. ค้นหา **ECPay** ใน **การเชื่อมต่อแพลตฟอร์มสนับสนุน**
4. นำ **HashKey การเชื่อมต่อ** และ **HashIV การเชื่อมต่อ** จาก **การตั้งค่าการเชื่อมต่อระบบ** มาใส่ในช่อง **Hash Key** และ **Hash IV** ตามลำดับ
5. คลิก **บันทึก**

![ECPay settings in Stream Toolkit](./ecpay-img/step3.png)

## ขั้นตอนที่ 4: ตั้งค่า URL การแจ้งเตือน

1. คัดลอก **URL แจ้งเตือนเบื้องหลัง** ของ ECPay

   ![Copy the notification URL](./ecpay-img/step4-1.png)

2. ในหลังบ้านร้านค้าของ ECPay ให้ค้นหา **เครื่องมือการชำระเงิน** → **การชำระเงินของสตรีมเมอร์**

   ![Streamer payment in payment tools menu](./ecpay-img/step4-2.png)

3. วาง **URL แจ้งเตือนเบื้องหลัง** ลงในช่อง **URL ส่งกลับการแจ้งเตือนเมื่อชำระเงินสำเร็จ**

   ![Payment completion notification URL field](./ecpay-img/step4-3.png)

4. คลิก **บันทึกการตั้งค่า**

## คำถามที่พบบ่อย

**Q: ไม่เห็น "การตั้งค่าระบบ" หลังจากเข้าสู่ระบบ?**
บัญชีของคุณอาจยังดำเนินการตรวจสอบไม่เสร็จสิ้น โปรดไปที่ "การจัดการข้อมูลร้านค้า" เพื่อตรวจสอบสถานะ

**Q: HashKey เปิดเผยเป็นสาธารณะได้ไหม?**
ไม่ได้ครับ HashKey และ HashIV เป็นคีย์ส่วนตัว โปรดอย่าแชร์ภาพหน้าจอหรือโพสต์ในที่สาธารณะ

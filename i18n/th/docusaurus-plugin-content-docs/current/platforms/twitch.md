# Twitch ตั้งค่า

Stream Toolkit ใช้การตรวจสอบสิทธิ์ผ่าน OAuth เพื่อเชื่อมต่อบัญชี Twitch ของคุณ ดังนั้นจึงไม่จำเป็นต้องกรอกคีย์ด้วยตนเอง

## ขั้นตอนที่ 1: เชื่อมต่อบัญชี Twitch

1. เปิด Stream Toolkit
2. คลิกที่ **ตั้งค่า** ในเมนูด้านซ้ายล่าง
3. ค้นหา **Twitch** ในการเชื่อมต่อแพลตฟอร์ม แล้วคลิก **เชื่อมต่อบัญชี**
4. เบราว์เซอร์จะเปิดหน้าการตรวจสอบสิทธิ์ของ Twitch โดยอัตโนมัติ
5. ลงชื่อเข้าใช้แล้วคลิก **อนุญาต**

หลังจากเสร็จสิ้นการตรวจสอบสิทธิ์ Stream Toolkit จะรับคีย์การเข้าถึงที่จำเป็นโดยอัตโนมัติ

## คำถามที่พบบ่อย

**Q: หลังจากอนุญาตแล้วไม่มีการตอบสนอง?**
ตรวจสอบให้แน่ใจว่าเบราว์เซอร์ไม่ได้บล็อกหน้าต่างป๊อปอัป Stream Toolkit จะเปิดหน้าต่างใหม่เพื่อดำเนินการตรวจสอบสิทธิ์ให้เสร็จสิ้น

**Q: จำเป็นต้องเป็นพันธมิตร (Partner) หรือรอยัลตี้ (Affiliate) ของ Twitch ถึงจะใช้ได้ไหม?**
ไม่จำเป็นครับ บัญชีทั่วไปก็สามารถใช้ฟังก์ชันแชทได้ ส่วนเป้าหมายการซับสคริปต์/Bits จำเป็นต้องเปิดใช้งานฟังก์ชันซับสคริปต์ก่อน

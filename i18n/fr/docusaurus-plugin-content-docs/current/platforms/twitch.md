# Twitch Réglages

Stream Toolkit utilise l'autorisation OAuth pour connecter votre compte Twitch, il n'est donc pas nécessaire de saisir manuellement une clé.

## Étape 1 : Connecter le compte Twitch

1. Ouvrez Stream Toolkit
2. Cliquez sur **Réglages** dans le menu en bas à gauche
3. Trouvez **Twitch** sous Connexions de plateformes et cliquez sur **Connecter le compte**
4. Le navigateur ouvrira automatiquement la page d'autorisation Twitch
5. Connectez-vous et cliquez sur **Autoriser**

Une fois l'autorisation accordée, Stream Toolkit obtiendra automatiquement la clé d'accès requise.

## Questions fréquentes

**Q : Pas de réaction après l'autorisation ?**
Vérifiez que votre navigateur ne bloque pas les fenêtres pop-up. Stream Toolkit ouvrira une nouvelle fenêtre pour terminer l'autorisation.

**Q : Doit-on être Partenaire ou Affilié Twitch pour l'utiliser ?**
Non. Les comptes standards peuvent également utiliser les fonctionnalités du chat. Les objectifs d'abonnements/Bits nécessitent l'activation des fonctionnalités d'abonnement.

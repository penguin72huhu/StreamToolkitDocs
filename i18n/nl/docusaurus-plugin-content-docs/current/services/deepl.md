# DeepL API-sleutel verkrijgen

De DeepL API-sleutel wordt gebruikt voor automatische vertaling van de volgende functies:
- **Spraakvertaling** — Automatische vertaling na omzetting van spraak naar tekst
- **Chat** — Automatische vertaling van berichten van kijkers

## Stap 1: Log in op uw DeepL-account

Ga naar [DeepL](https://www.deepl.com) en log in op uw account. Als u nog geen account heeft, moet u zich eerst registreren.

## Stap 2: Ga naar Account-instellingen

Klik op het **profielicoon** in de rechterbovenhoek en selecteer **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Stap 3: Overschakelen naar het tabblad API Keys & limits

Klik op het tabblad **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Stap 4: Maak een nieuwe API Key aan

1. Klik op **Create key +**
2. **Name your key**: Voer een willekeurige naam in (bijv. `Stream Toolkit`)
3. **Permissions**: Selecteer **All access**
4. Klik op **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Stap 5: Kopiëren en plakken in de App

1. Kopieer de gegenereerde API Key
2. Ga terug naar Stream Toolkit en plak het in het bijbehorende **DeepL API Key**-veld

![Paste API Key](./deepl-api-img/step5.png)

## Veelgestelde vragen

**Q: Heeft de gratis proefversie van DeepL gebruikslimieten?**
Ja. De gratis proefversie biedt een quotum van 1.000.000 tekens en is beperkt tot één maand. Voor voortgezet gebruik van hoogwaardige vertalingen kunt u zich abonneren op een betaald DeepL-abonnement.

**Q: Wat moet ik doen als mijn API Key is gelekt?**
Ga terug naar DeepL Account → API Keys & limits, verwijder de oude Key en maak een nieuwe aan.

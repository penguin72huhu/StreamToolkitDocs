# Ottenimento della chiave API DeepL

La chiave API DeepL viene utilizzata per la traduzione automatica delle seguenti funzioni:
- **Traduzione vocale** — Traduzione automatica dopo la conversione del parlato in testo
- **Chat** — Traduzione automatica dei messaggi degli spettatori

## Passaggio 1: Accedi al tuo account DeepL

Vai su [DeepL](https://www.deepl.com) e accedi al tuo account. Se non hai ancora un account, devi prima registrarti.

## Passaggio 2: Vai alle impostazioni di Account

Clicca sull'**icona del profilo** nell'angolo in alto a destra e seleziona **Account**.

![Open Account settings](./deepl-api-img/step2.png)

## Passaggio 3: Passa alla scheda API Keys & limits

Clicca sulla scheda **API Keys & limits**.

![API Keys & limits tab](./deepl-api-img/step3.png)

## Passaggio 4: Crea una nuova API Key

1. Clicca su **Create key +**
2. **Name your key**: Inserisci un nome a piacere (es. `Stream Toolkit`)
3. **Permissions**: Seleziona **All access**
4. Clicca su **Create Key**

![Create API Key](./deepl-api-img/step4.png)

![API Key created](./deepl-api-img/step4-1.png)

## Passaggio 5: Copia e incolla nell'App

1. Copia la API Key generata
2. Torna a Stream Toolkit e incollalo nel campo **DeepL API Key** corrispondente

![Paste API Key](./deepl-api-img/step5.png)

## Domande frequenti

**Q: La versione di prova gratuita di DeepL ha limiti di utilizzo?**
Sì. La versione di prova gratuita offre una quota di 1.000.000 di caratteri ed è limitata a un mese. Per continuare a utilizzare traduzioni di alta qualità, iscriviti a un piano a pagamento DeepL.

**Q: Cosa fare se la mia API Key viene compromessa?**
Torna a DeepL Account → API Keys & limits, elimina la vecchia Key e creane una nuova.

# Installazione e avvio

## Requisiti di sistema

- Windows 10 / 11 (64-bit)
- Connessione Internet

## Download e installazione

1. Vai alla [pagina di Steam](https://store.steampowered.com/app/4828880/Stream_Toolkit/) per acquistare Stream Toolkit
2. Fai clic su "Installa" su Steam e attendi il completamento del download
3. Avvia Stream Toolkit da Steam

## Primo avvio

Dopo l'avvio, vedrai l'interfaccia di controllo principale, con il menu a sinistra che elenca tutte le funzioni.

Ogni funzione di Stream Toolkit ha un URL locale corrispondente (ad esempio, `http://127.0.0.1:7272/donate`). Aggiungendo una "Fonte browser" in OBS e inserendo quell'URL, puoi far apparire la schermata della função nella diretta streaming.

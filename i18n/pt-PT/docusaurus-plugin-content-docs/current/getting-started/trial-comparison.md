# Comparação entre Versão de Teste e Versão Completa

O Stream Toolkit oferece uma **versão de teste gratuita** e uma **versão completa**. A versão de teste não tem limite de tempo e permite o uso permanente da maioria das funcionalidades interativas para uma experiência completa; a versão completa desbloqueia integrações de pagamento, Steam Workshop, personalização avançada e remove vários limites de quantidade.

:::info Sobre o Upgrade
A versão de teste e a versão completa são duas aplicações independentes no Steam. Após adquirir a versão completa, ela deve ser instalada separadamente, e as definições e registos não são partilhados entre si.
:::

# Twitch 設定

Stream Toolkit は OAuth 認証を使用して Twitch アカウントと連携するため、キーを手動で入力する必要はありません。

## ステップ 1：Twitch アカウントを連携

1. Stream Toolkit を起動します
2. 左下のメニューの **設定** をクリックします
3. プラットフォーム連携から **Twitch** を見つけ、**アカウントを連携** をクリックします
4. ブラウザが自動的に Twitch の認証ページを開きます
5. ログインして **認証** をクリックします

認証が完了すると、Stream Toolkit は必要なアクセスキーを自動的に取得します。

## よくある質問

**Q：認証した後に反応がありません。**
ブラウザでポップアップウィンドウがブロックされていないか確認してください。Stream Toolkit は新しいウィンドウを開いて認証を完了します。

**Q：Twitch パートナーやアフィリエイトである必要がありますか？**
がいいえ、その必要はありません。一般のアカウントでもチャット機能は利用可能です。サブスク/Bits 目標にはサブスクリプション機能の解放が必要です。

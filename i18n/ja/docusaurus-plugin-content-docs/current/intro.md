# Stream Toolkit 紹介

左側のメニューは機能別に分類されています。初心者は [クイックスタート](./getting-started/installation) から読み始めることをお勧めします。

---

Stream Toolkit は配信者向けのオールインワン配信ツールです。複数のサイトやツールを切り替える手間をなくします。

- **リアルタイム翻訳**： チャット翻訳と音声翻訳により、世界中の視聴者と簡単に交流できます。
- **豊富な機能**： 投票、抽選、目標バー、支援通知、メディアボードをひとつに統合。
- **カスタマイズ**： CSS カスタマイズと Steam Workshop テーマに対応し、自由にデザインを変更できます。
- **簡単セットアップ**： Overlay URL をワンクリックで生成し、OBS に貼り付けるだけで利用できます。

---

## 🌍リアルタイムチャット翻訳

Twitch と YouTube のチャットを統合。

外国語のメッセージを自動翻訳し、世界中の視聴者とより簡単に交流できます。

アバター、バッジ、プラットフォームアイコン表示に対応し、CSS カスタマイズや Steam Workshop テーマ共有も利用できます。

## 🎙️ 翻訳字幕

ホットキーを押して録音を開始します。

発言を指定した言語に翻訳し、配信画面に表示することで、異なる言語の視聴者にも内容を伝えられます。

## 🎨 Steam Workshop テーマ

コミュニティが作成したテーマをダウンロードしたり、自分だけの Overlay スタイルを作成できます。

チャット、目標バー、支援通知、タイマーまで自由に外観やアニメーションを調整でき、配信画面を個性的に演出できます。

Steam Workshop での共有とダウンロードに対応。

## 🎮 配信ツールをひとつに統合

搭載機能：

- ルーレット
- リアルタイム投票 / 抽選
- Twitch 目標バー
- 支援目標バー / 支援通知 / 支援履歴ボード
- タイマー
- 時計
- リクエスト曲
- メディアボード（画像 / 動画 / 効果音）
- エモートウォール
- ハイライトマーカー

複数のプラグインやウェブツールを導入する必要はありません。

## ⚡ OBS 向け設計

すべての機能に専用の OBS URL を用意。設定後はコピー＆ペーストするだけで利用できます。

チャット、目標バー、タイマー、支援通知などの機能は個別に動作し、不要な機能はパフォーマンスに影響しません。

複数のサイトやツールを切り替える時間を減らし、配信そのものに集中できます。

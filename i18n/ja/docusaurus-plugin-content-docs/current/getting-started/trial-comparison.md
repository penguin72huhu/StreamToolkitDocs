# 試用版 / 製品版の比較

Stream Toolkit には**無料試用版**と**製品版**があります。試用版には時間制限がなく、大部分のインタラクティブ機能を永久に使用して十分に体験できます。製品版では、決済連携、Steam ワークショップ、高度なカスタマイズ、および各種数量制限の解除がアンロックされます。

# インストールと起動

## システム要件

- Windows 10 / 11 (64-bit)
- インターネット接続

## ダウンロードとインストール

1. [Steam ページ](https://store.steampowered.com/app/4828880/Stream_Toolkit/) に移動して Stream Toolkit を購入します
2. Steam で「インストール」をクリックし、ダウンロードが完了するまで待ちます
3. Steam から Stream Toolkit を起動します

## 初回起動

起動するとメインコントロール画面が表示され、左側のメニューにすべての機能が一覧表示されます。

Stream Toolkit の各機能には、対応するローカル URL（例：`http://127.0.0.1:7272/donate`）があります。OBS で「ブラウザソース」を追加してその URL を入力すると、配信画面に機能の画面を表示させることができます。

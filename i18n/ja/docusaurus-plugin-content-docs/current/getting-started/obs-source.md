# OBS でソースを追加する方法

Stream Toolkit のほとんどの機能は、配信画面に表示するために OBS で「ブラウザソース」を追加する必要があります。

## ステップ 1: 機能の URL をコピーする

Stream Toolkit では、各機能のページに対応するローカル URL が次の形式で表示されます：

```
http://127.0.0.1:7272/feature-name
```

追加したい機能の URL をコピーします。

## ステップ 2: OBS でブラウザソースを追加する

1. OBS の **ソース** セクションで、下部にある **+** ボタンをクリックします
2. **ブラウザ** を選択します
3. **新規作成** を選択し、ソースの名前（例：「支援目標バー」）を入力して **OK** をクリックします
4. **URL** フィールドにステップ 1 でコピーした URL を貼り付けます
5. 必要に応じて **幅** と **高さ** を調整します
6. **OK** をクリックします

## 以下は「OBS を介して音声を制御する」にチェックを入れる必要がある機能です

音声出力が含まれるため、配信に音声を正しく出すには、ブラウザソースの設定で **OBS を介して音声を制御する** にチェックを入れる必要があります：

- **メディアボード** — 動画、サウンド
- **支援通知**
- **リクエスト曲**

チェック方法：
1. OBS のソース一覧で、該当するブラウザソースを右クリックします
2. **プロパティ** を選択します
3. **OBS を介して音声を制御する** を見つけてチェックを入れます
4. **OK** をクリックします

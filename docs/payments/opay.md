# 歐付寶設定

本教學說明如何取得歐付寶的 **HashKey** 與 **HashIV**，並填入 Stream Toolkit。

## 前置需求

- 已申請歐付寶商店帳號（需實名驗證）
- 已開通「信用卡定期定額」或「超商代碼」等收款功能

## 步驟一：登入歐付寶廠商後台

1. 前往 [歐付寶官網](https://www.opay.tw/) 並登入帳號
2. 登入後點選右上角進入廠商後台

:::note
如果還沒有歐付寶帳號，需要先完成商店申請與身分驗證。
:::

## 步驟二：進入系統開發管理

1. 登入後，點選上方選單的 **系統開發管理**
2. 選擇 **廠商資料設定**

## 步驟三：找到 HashKey 與 HashIV

在「廠商資料設定」頁面，往下捲動找到 **串接資訊** 區塊：

| 欄位 | 說明 |
|------|------|
| **廠商代號（MerchantID）** | 你的商店唯一識別碼 |
| **HashKey** | 加密金鑰，請勿外洩 |
| **HashIV** | 加密初始向量，請勿外洩 |

{/* TODO: 截圖 — 串接資訊 */}

## 步驟四：填入 Stream Toolkit

1. 開啟 Stream Toolkit
2. 點選左側選單的 **贊助設定**
3. 選擇付款方式為 **歐付寶**
4. 將以下資訊填入對應欄位：
   - 廠商代號（MerchantID）
   - HashKey
   - HashIV
5. 點擊 **儲存**

## 常見問題

**Q：找不到「系統開發管理」選單？**
代表帳號還未通過審核，或尚未開通相關收款功能。請聯繫歐付寶客服。

**Q：HashKey 可以公開嗎？**
不行。HashKey 和 HashIV 屬於私密金鑰，請勿截圖分享或貼到公開場合。

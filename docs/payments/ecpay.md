# 綠界設定

本教學說明如何取得綠界的 **HashKey** 與 **HashIV**，並填入 Stream Toolkit。

## 前置需求

- 已申請綠界科技特約商店帳號
- 已通過身分審核

## 步驟一：登入綠界廠商後台

1. 前往 [綠界廠商後台](https://vendor.ecpay.com.tw/Account/Login)
2. 輸入帳號密碼登入

## 步驟二：進入 API 串接設定

1. 點選上方選單的 **系統開發管理**
2. 選擇 **系統介接設定**

## 步驟三：找到 HashKey 與 HashIV

在「系統介接設定」頁面可以看到：

| 欄位 | 說明 |
|------|------|
| **特店編號（MerchantID）** | 你的商店唯一識別碼 |
| **HashKey** | 加密金鑰，請勿外洩 |
| **HashIV** | 加密初始向量，請勿外洩 |

:::warning 注意
正式環境與測試環境的 HashKey/HashIV 不同，請確認使用**正式環境**的金鑰。
:::

## 步驟四：填入 Stream Toolkit

1. 開啟 Stream Toolkit
2. 點選左側選單的 **贊助設定**
3. 選擇付款方式為 **綠界**
4. 填入對應欄位：
   - 特店編號（MerchantID）
   - HashKey
   - HashIV
5. 點擊 **儲存**

## 常見問題

**Q：登入後看不到「系統開發管理」？**
帳號可能尚未完成審核。請至「廠商資料管理」確認審核狀態。

**Q：測試環境的金鑰可以用嗎？**
不行。測試環境的交易不會真正收到款項。請務必使用正式環境金鑰。

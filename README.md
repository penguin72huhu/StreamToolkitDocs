# Stream Toolkit Docs

Official documentation for [Stream Toolkit](https://store.steampowered.com/app/4828880/Stream_Toolkit/) — an all-in-one streaming toolbox for content creators on Twitch and YouTube.

Covers setup guides for payment integrations (PayPal, Stripe, O'Pay, ECPay), platform connections (Twitch, YouTube), and third-party services (DeepL).
